# Lightweight Computation Single-Image Fog Removal based on a New Improved Adaptive Dark Channel Prior
MATLAB implementation of paper "Lightweight Computation Single-Image Fog Removal based on a New Improved Adaptive Dark Channel Prior" (IEEE T-ITS 2025)

## Getting Started
### Requirements
MATLAB with Image Processing Toolbox
### Running the Code
1. Clone this repository
2. Open the repository in MATLAB
3. Run `demo('images/image_name.extension')` to test with provided sample images
4. The dehazed image will be saved in the parent directory
